Learn NodeJS (1)

学习Node.js （一）

最近业余学习一下Node.js，记录一下学习路径。

初学乍练

第一步 Node初学指南（中文版）

跟随指南做练习，其中引用到的文章，从学习效率的角度考虑，说明如下：

  • Understanding node.js 必读
  • Martin Fowler关于依赖注入的大作 推荐阅读
  • Steve Yegge的大作名词王国中的死刑 可略过不读，不过特想买一本Steve的A Programmer’s Rantings，想一睹吐槽大师的风采
  • Javascript中的对象与传统面向对象语言中对象的区别 已略过不读，理解概念即可
  • 理解Node.js的事件轮询 必读，作者自己有一本免费的Node小书，具体见下文

通过第一步，对NodeJS的基本概念、机制有了初步了解，能写一个Web服务器程序处理基本请求。

第二步 寻找更多免费靠谱资源

互联网资源遍地，但是靠谱又免费的不多啊！下面使用排除法，避免你掉进相同的小水沟。:)

推荐列表

  • nodetuts.com 多个Node的视频讲座，12/13年的，推荐。
  • HowtoNode 不系统，不是针对入门级选手，应该是针对中高级Node开发者的，可作日后参考。
  • Mixu’s Node小书 比较用心的作者，保持更新，推荐

不推荐列表

  • Udemy的视频课程 只有少部分课程免费，不能系统学习，不推荐。
  • Mastering Node 貌似以前比较知名的Node开源电子书，但是2年没有更新，章节不全，不推荐。
  • Felix’s beginner’s guide 可以浏览一下，但不是教程，可作为编程参考。

参考

  • Stackoverflow关于学习Node的资源问答 不服不行，Stackoverflow真是码农的好朋友，赞。这是一份资源列表，还需要进一步根据学习的需要过滤。我会在学习NodeJS后续的文章中陆续引用其中的资源并给出评价。
  • developerWorks提供的NodeJS的学习资料汇总
  • 译言文章NodeJS学习的最好资源

第二步，通过靠谱的资源继续推进，进入高级入门阶段。:D
